# Banking Domain – Manual Testing Project

## Overview
This repository contains a **manual testing project for a Banking Application**, focused on the **Fund Transfer module**.  
The project demonstrates requirement analysis, test case creation, defect tracking, and test metrics using **Jira and GitHub**.

---

## Domain & Module
- **Domain:** Banking  
- **Module:** Fund Transfer

---

## Tools Used
- Jira – Test case management & defect tracking  
- Git & GitHub – Version control and project repository  
- Excel / CSV – Test case and defect exports  

---

## Test Artifacts
- Software Requirement Specification (SRS)
- Test Scenarios
- Test Cases
- Defect Reports
- Test Metrics

---
